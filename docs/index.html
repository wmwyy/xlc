闸孔净宽计算器（A.0.1~A.0.3）
C# / WPF 桌面应用 · 按附录 A 公式计算闸孔净宽

概览

本工具包含三个典型工况：平底闸门 A.0.1、高底闸门 A.0.2、潜没闸门 A.0.3；界面多标签页，支持示例一键填充，实时显示中间参数。

主要功能

  • 输入参数与国标附录 A 公式一致，系数可自定义。
  • 中间参数实时计算：sigma、epsilon、mu0、mu、epsilon'、lambda、sigma'。
  • 内置表 A.0.3 插值，超界自动取端点。
  • 示例按钮：一键填入 README 中的示例值。

运行/构建

  1. 安装 .NET 8 SDK。
  2. 在仓库根目录执行：dotnet run --project GateCalculator
  3. 构建：dotnet build GateCalculator

这是桌面应用，GitHub Pages 仅用于展示说明与下载链接，不提供在线运行。

文件与结构

  • GateCalculator/MainWindow.xaml(.cs)：UI 布局、示例填充、简图显示与缩放。
  • GateCalculator/Services/Calculators.cs：三种工况计算公式与插值表。
  • Assets/A01.png, A02.png, A03.png：三种工况示意图，支持界面右侧显示与缩放。
  • README.md：公式、示例值、运行说明。

使用步骤（桌面端）

  1. 确保已安装 .NET 8 SDK。
  2. 克隆仓库：git clone https://github.com/wmwyy/zkjk.git
  3. 运行应用：dotnet run --project GateCalculator
  4. 在界面选择工况、可先点击“填充示例”，再修改参数后计算。
  5. 右侧可查看对应工况的示意图，使用滑杆缩放。
